## списки

> чтобы добавить ненумерованные списки, необходимо пункты выделить звёздочкой, например
* элемент 1
* элемент 2
* элемент 3

Для создания нумерованного списка достаточно перед пронумеровать пункты. К примеру так:
1. пункт 1
2. пункт 2
3. пункт 3
